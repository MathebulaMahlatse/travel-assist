##### Running the project locally
* Install Git
* Install Node using NVM (Node Version Manager)
* Clone 
```
    git clone https://github.com/MathebulaMahlatse/travel-assist.git
```
* Install node modules 
```
    npm install
```

* Run unit tests
```
    npm test
```

* Run local app
```
    npm start
```
